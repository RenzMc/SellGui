# SellGui

SellGui adalah plugin yang memudahkan pemain untuk menjual item di server Pocketmine. Dengan antarmuka pengguna yang ramah, pemain dapat dengan mudah menjelajahi inventaris mereka dan memilih item yang ingin mereka jual.

Fitur utama SellGui meliputi:
1. **Antarmuka Pengguna yang Ramah**: Antarmuka yang mudah dinavigasi membuat proses penjualan menjadi lebih cepat dan efisien.
3.  **SellOres**: Menjual semua bijih yang Anda miliki dengan sekali klik, memungkinkan pemain untuk mendapatkan uang dengan cepat dari tambang mereka.

4. **SellAll**: Mengizinkan pemain untuk menjual seluruh inventaris item mereka hanya dengan satu klik, secara signifikan meningkatkan efisiensi dalam menjual item.

5. **SellInventory**: Memungkinkan pemain untuk menjual seluruh isi inventaris mereka ke dalam toko dengan cepat dan mudah.

6. **SellHand**: Dengan fitur ini, pemain dapat langsung menjual item yang mereka pegang dalam tangan mereka.
